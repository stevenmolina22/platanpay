## Local Development

Install dependencies:

```bash
npm install
```

Run the Next.js app:

```bash
npm run dev
```

Open `http://127.0.0.1:3000`.

Mock/demo mode:

```bash
PLATANDPAY_AGENT_MOCK=1 npm run dev
```

Real agent mode requires `ANTHROPIC_API_KEY` in `.env`. `FIRECRAWL_API_KEY` is optional for broader live product search.
