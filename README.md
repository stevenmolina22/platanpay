# PlatandPay 🍌

Agente de compras supervisado para Argentina. Parte del sistema **Brújula** (Paper v1.0).

## Arrancar

```bash
npm install
cp .env.example .env  # poner tu ANTHROPIC_API_KEY
npm run dev:cli       # chat por terminal
npm run dev:server    # API en http://localhost:3000
```

Docs extensos en `/docs/architecture.md`.
